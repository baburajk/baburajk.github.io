REST vs Non REST

/dvlbr-xxxx/certs/updatecert.jsp&action=rename&newname=abc.xyz There is no Resource here and it's an instruction. https://restfulapi.net/resource-naming/

ReST

POST /dvlbr-xxxx/certs -d@payload.json Here command points to a resource.

PUT vs POST, if you use curl -X PUT vs curl -X POST

PUT replaces an existing resource or create a resource it it doesn't exist. POST creates a new resource PUT is idempotent vs POST is not idempotent. https://restfulapi.net/rest-put-vs-post/
